# !commands

<PlatformBadges supported={[ 'Twitch', 'YouTube', 'Trovo', 'Kick' ]} />

## Overview

The `!commands` command provides viewers with a link to the streamer's public StreamElements commands page. This page lists all enabled default and custom commands for the channel, making it easy for viewers to discover available interactions.

## Usage

To get the link to the commands page, simply type the command in chat:

```streamelements
!commands
```

## Examples

<ChatExample messages={[
  { persona: 'user', message: '!commands' },

]} />

## Configuration

*   This command is enabled by default.
*   The public commands page only shows commands that are currently enabled.
*   The list updates automatically based on your command settings in the StreamElements dashboard.
