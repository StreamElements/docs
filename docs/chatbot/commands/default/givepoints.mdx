# !givepoints

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!givepoints` command allows viewers to transfer their loyalty points to another user in the chat. This enables community interactions like gifting points or pooling resources.

## Usage

To transfer points to another user:

```streamelements
!givepoints <username> <amount>
```

## Examples

<ChatExample messages={[

## Parameters

*   `<username>` (Required): The exact username of the recipient.
*   `<amount>` (Required): The number of points to transfer. Must be a positive integer, and the sender must have sufficient points.

## Configuration

*   The `!givepoints` command requires the **Loyalty system** to be active. Enable it in your **StreamElements Dashboard** under `Loyalty` -> `Loyalty Settings`.
*   The ability for users to give points can be enabled/disabled within the Loyalty Settings.
*   Streamers can set a minimum user level required to use the command and a cooldown period in the command's settings under `Chatbot` -> `Chat Commands` -> `Default Commands`.
