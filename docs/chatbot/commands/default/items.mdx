# !items

<PlatformBadges supported={[ 'Twitch', 'YouTube', 'Trovo', 'Kick' ]} />

## Overview

The `!items` command provides viewers with a direct link to the channel's StreamElements loyalty store page. This page displays all currently available items that can be redeemed using loyalty points.

## Usage

To get the link to the store page, simply type the command in chat:

```streamelements
!items
```

## Examples

<ChatExample messages={[
  { persona: 'user', message: '!items' },

## Configuration

*   The linked store page only shows items that are currently enabled and redeemable.
