# !join

<PlatformBadges supported={[ 'Twitch', 'YouTube', 'Trovo', 'Kick' ]} />

## Overview

The `!join` command allows viewers to enter an **active raffle** in the chat. Viewers use this command to participate in raffles started by moderators using either [`!sraffle`](sraffle.mdx) or [`!raffle`](raffle.mdx).

## Usage

To enter the currently active raffle, simply type:

```streamelements
!join
```

There are **no parameters** for this command.

## Examples

**Entering a raffle (`!sraffle`):**
<ChatExample messages={[
  { persona: 'moderator', message: '!sraffle' },

  { persona: 'user', message: '!join' },

## Parameters

This command does not accept any parameters.

## Configuration

*   The **Raffle module** must be enabled in your StreamElements dashboard (`Chatbot` -> `Modules` -> `Raffle`).

## Related Commands

*   [`!sraffle`](sraffle.mdx): Starts a standard, single-entry raffle.
*   [`!raffle`](raffle.mdx): Starts a points-based raffle where viewers spend points to enter (entry via `!ticket` or `!join`).
*   [`!cancelraffle`](cancelraffle.mdx): Cancels the active raffle.
