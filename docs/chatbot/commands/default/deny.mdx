# !deny

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!deny` command allows a user to reject an **incoming** duel request initiated by another user with the [`!duel`](duel.mdx) command.

## Usage

To reject a pending incoming duel request, type the command in chat:

```streamelements
!deny
```

## Configuration

The `!deny` command is part of the **Duel module**. This module must be enabled in your StreamElements chatbot settings under `Chatbot` -> `Modules` for the command to function.

## Related Commands

*   [`!duel`](duel.mdx): Initiate a duel with another user.
*   [`!accept`](accept.mdx): Accept an incoming duel request.
*   [`!cancelduel`](cancelduel.mdx): Cancel an *outgoing* duel request you initiated.

## Aliases

By default, there are no aliases for the `!deny` command. However, moderators can create custom aliases if needed.
