# !next

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!next` command (aliased as `!nextsong`) allows anyone in chat to view details about the upcoming song in the StreamElements media request queue. It typically shows the song title, duration, requester, and a link.

## Usage

To see the next song in the queue:

```streamelements
!next
```

Or using the alias:

```streamelements
!nextsong
```

## Configuration

*   This command requires the **Songrequest** module to be enabled and configured in your StreamElements Dashboard (`Chatbot` -> `Modules`).
*   The information displayed depends on the data available from YouTube for the requested track.

## Related Commands

*   [`!song`](song.mdx): Shows details about the *currently playing* song.
*   [`!songqueue`](songqueue.mdx): Provides a link to the full media request queue.
*   [`!songrequest`](songrequest.mdx): Allows users to request songs to add to the queue.
*   [`!skip`](skip.mdx): (Moderator+) Skips the currently playing song.
*   [`!removesong`](removesong.mdx): (Moderator+) Removes a specific song from the queue.
