# !song

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!song` command allows viewers to retrieve information about the currently playing song in the StreamElements media request queue. This command is useful for viewers who want to know the title, artist, and requester of the current song.

## Usage

To use the `!song` command, simply type it in the chat:

```streamelements
!song
```

## Examples

<ChatExample messages={[
  { persona: 'user', message: '!song' },
  { persona: 'bot', message: '@User, current song: DrakeVEVO - Drake, 21 Savage - Rich Flex (Audio), requested by styler https://youtu.be/I4DjHHVHWAE' }
]} />

<ChatExample messages={[

]} />

## Configuration

*   This command requires the **Media Request** module to be enabled and configured in your StreamElements Dashboard (`Chatbot` -> `Modules`).
*   The information displayed depends on the media source (e.g., YouTube, Spotify) and the data available for the requested track.
