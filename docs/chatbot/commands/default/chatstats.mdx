# !chatstats

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!chatstats` command provides a direct link to the streamer's detailed chat statistics page on StreamElements. This allows anyone in chat to easily access information about chat activity, top chatters, command usage, and more.

## Usage

To get the link to the chat statistics page, simply type the command in chat:

```streamelements
!chatstats
```
