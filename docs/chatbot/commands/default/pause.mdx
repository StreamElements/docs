# !pause

<PlatformBadges supported={[ 'Twitch', 'YouTube', 'Trovo', 'Kick' ]} />

## Overview

The `!pause` command temporarily stops the playback of the current item in the StreamElements media request queue. The queue itself remains intact, and playback can be resumed later.

## Usage

To pause the currently playing media:

```streamelements
!pause
```

**Important:** By default, only users with **Moderator** permission level or higher can use this command.

## Configuration

*   The command's permission level can be adjusted under `Chatbot` -> `Chat Commands` -> `Default Commands`.

## Related Commands

*   [`!play`](play.mdx): Resumes media playback after it has been paused.
*   [`!skip`](skip.mdx): Skips the currently playing media item.
*   [`!volume`](volume.mdx): Adjusts the volume of the media player.
*   [`!songrequest`](songrequest.mdx): Adds a song/video to the queue.
*   [`!songqueue`](songqueue.mdx): Provides a link to the media queue page.
