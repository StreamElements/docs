# !enablesfx

<PlatformBadges supported={[ 'Twitch', 'YouTube', 'Trovo', 'Kick' ]} />

## Overview

The `!enablesfx` command allows streamers and moderators to quickly re-enable all **sound effect** (SFX) items in the StreamElements loyalty store that were previously disabled (e.g., using `!disablesfx`).

## Usage

To enable all SFX store items, use the following command in chat:

```streamelements
!enablesfx
```

**Important:** By default, only users with **Moderator** permission level or higher can use this command.

## Examples

<ChatExample messages={[
  { persona: 'moderator', message: '!enablesfx' },

<ChatExample messages={[

]} />

## Configuration

*   The `!enablesfx` command requires the **Stream Store** module to be active and configured with SFX items. Manage this in your **StreamElements Dashboard** under `Loyalty` -> `Stream Store`.
*   The command's permission level can be adjusted under `Chatbot` -> `Chat Commands` -> `Default Commands`.

## Related Commands

*   [`!disablesfx`](disablesfx.mdx): Disables all sound effect (SFX) items in the loyalty store.
*   [`!closestore`](closestore.mdx): Disables all *regular* (non-SFX) items in the loyalty store.
*   [`!openstore`](openstore.mdx): Enables all *regular* (non-SFX) items in the loyalty store.
*   [`!redeem`](redeem.mdx): Allows viewers to redeem store items.
