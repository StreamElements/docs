# !accept Command

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!accept` command allows users to accept pending duel requests initiated by the [`!duel`](duel.mdx) command.

## Usage

To use the `!accept` command, simply type it in the chat when you have a pending duel request.

```streamelements
!accept
```

## Examples

<ChatExample
    messages={[

    ]}
/>

## Parameters

This command does not take any parameters.

## Configuration

The `!accept` command is part of the **Duel module**. This module must be enabled in your StreamElements chatbot settings under `Chatbot` -> `Modules` for the command to function.

## Related Commands

*   [`!duel`](duel.mdx): Initiate a duel with another user.
*   [`!cancelduel`](cancelduel.mdx): Cancel an outgoing duel request.
*   [`!deny`](deny.mdx): Deny an incoming duel request.
