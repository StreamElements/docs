# !giveaway

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!giveaway` command allows anyone in chat to check the status of the currently active channel giveaway. If a giveaway is running, the command provides its name and a direct link for viewers to enter.

**Note:** This command *checks* the status of an active giveaway. To *start* or manage giveaways, use the StreamElements Dashboard Giveaway Module.

## Usage

To check the active giveaway details, simply type the command in chat:

```streamelements
!giveaway
```

## Configuration

*   The `!giveaway` command requires a giveaway to be running. Manage giveaways in your **StreamElements Dashboard** under `Loyalty` -> `Giveaways`.
*   The command response reflects the settings of the currently active giveaway (name, entry requirements) configured by the streamer/moderator.
