# !kappagen

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!kappagen` command triggers a visual explosion of emotes on the stream overlay using the **StreamElements Kappagen widget**. This adds a fun, dynamic visual effect to the broadcast.

## Usage

To trigger the emote explosion:

```streamelements
!kappagen
```

## Examples

<ChatExample messages={[
  { persona: 'user', message: '!kappagen' }
  // No chat output from the bot. A visual effect occurs on the overlay.
]} />

## Configuration

*   The **Kappagen widget** must be added to an active StreamElements overlay.
*   The `!kappagen` command must be enabled in your StreamElements Dashboard under `Chatbot` -> `Chat Commands` -> `Default Commands`.
*   You can customize the Kappagen widget settings (emotes used, intensity, duration, cooldown, cost, permissions) within the overlay editor and the command settings.
