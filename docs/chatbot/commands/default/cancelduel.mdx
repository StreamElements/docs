# !cancelduel

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!cancelduel` command allows a user to cancel an **outgoing** duel request they initiated with the [`!duel`](duel.mdx) command *before* the challenged user accepts or denies it.

## Usage

To cancel your pending outgoing duel request, simply type the command in chat:

```streamelements
!cancelduel
```

## Configuration

The `!cancelduel` command is part of the **Duel module**. This module must be enabled in your StreamElements chatbot settings under `Chatbot` -> `Modules` for the command to function.

## Related Commands

*   [`!duel`](duel.mdx): Initiate a duel with another user.
*   [`!accept`](accept.mdx): Accept an incoming duel request.
*   [`!deny`](deny.mdx): Deny an incoming duel request.
