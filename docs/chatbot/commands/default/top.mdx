# !top

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!top` command displays the top users in chat based on either loyalty points or watch time, depending on the parameter provided. It provides a quick glimpse of the leaderboard directly within the chat interface.

## Usage

To view a specific leaderboard:

```streamelements
!top <type>
```

If `<type>` is omitted, the bot typically lists the available leaderboard types.

## Parameters

*   `<type>` (Optional): The type of leaderboard to display. Common valid options are:
    *   `points`: Shows the top users based on loyalty points.
    *   `time` / `watchtime` / `alltime`: Displays the leaderboard based on total watch time (exact keyword may vary).
    *   `online`: Shows the top users based on watch time while the stream was live.
    *   `offline`: Displays the top users based on watch time while the stream was offline (if tracked).

    *(Check the output of `!top` without parameters or your Loyalty settings for the exact available types.)*

## Configuration

*   Requires the **Loyalty system** to be active (`Loyalty` -> `Loyalty Settings`).
*   The leaderboards displayed depend on the Loyalty module configuration (points tracking, watch time tracking enabled).
*   The name of the points currency (e.g., "ChannelPoints") is set in Loyalty Settings.
