# !hypecup

<PlatformBadges supported={[ 'Twitch' ]} />

## Overview

The `!hypecup` command allows streamers and moderators to clear the list of contributors displayed in the StreamElements HypeCup overlay widget.

## Usage

To clear the HypeCup overlay, use the following command in chat:

```streamelements
!hypecup clear
```

**Important:** By default, only users with **Moderator** permission level or higher can use this command.

## Examples

<ChatExample messages={[
  { persona: 'moderator', message: '!hypecup clear' },
  { persona: 'bot', message: '@Moderator, Successfully cleared HypeCup.' }
]} />

<ChatExample messages={[
  { persona: 'broadcaster', message: '!hypecup' }, // Missing the 'clear' parameter

## Parameters

*   `clear` (Required): This is the only valid parameter and triggers the action to clear the HypeCup overlay list.

## Configuration

*   This command interacts with the **HypeCup widget** in your StreamElements overlays. Ensure the widget is added to an active overlay.
*   The command's permission level can be adjusted under `Chatbot` -> `Chat Commands` -> `Default Commands`.
