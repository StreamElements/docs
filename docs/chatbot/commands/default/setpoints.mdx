# !setpoints

<PlatformBadges supported={[ 'Twitch', 'YouTube', 'Trovo', 'Kick' ]} />

## Overview

The `!setpoints` command allows streamers and moderators to set a specific number of loyalty points for a user in the chat. This command is useful for managing loyalty programs, rewarding viewers, or correcting point balances.

:::caution
Using this command will **overwrite** the user's current point balance. Use [`!addpoints`](addpoints.mdx) if you want to add to their balance instead.
:::

**Important:** By default, only users with **Moderator** permission level or higher can use this command.

## Usage

To set a user's point balance:

```streamelements
!setpoints <username> <points>
```

## Examples

<ChatExample messages={[

<ChatExample messages={[

## Parameters

*   `<username>` (Required): The username of the viewer whose points you want to set.
*   `<points>` (Required): The new amount of points to assign to the user.

## Configuration

*   This command requires the **Loyalty system** to be active (`Loyalty` -> `Loyalty Settings`).
*   The command's permission level can be adjusted under `Chatbot` -> `Chat Commands` -> `Default Commands`.

## Related Commands

*   [`!addpoints`](addpoints.mdx): Adds points to a user's balance without overwriting.
*   [`!givepoints`](givepoints.mdx): Allows users to transfer points.
*   [`!points`](points.mdx): Checks a user's current point balance.
*   [`!top`](top.mdx): View the leaderboard for points.
