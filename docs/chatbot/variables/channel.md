# $(channel)

Displays the name of the channel

#### Parameters

This variable does not take any parameters.

#### Example Output

```
styler
```

## $(channel.viewers)

Displays the channel’s current viewer count

#### Parameters

This variable does not take any parameters.

#### Example Input

```
$(channel) has $(channel.viewers) viewers
```

#### Example Output

```
onslaught has 100 viewers
```
