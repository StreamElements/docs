# $(math)

Outputs the result of a math.js expression.

#### Parameters

This variable takes a math.js expression as a parameter. You can also use math.js functions like `round()`, `floor()`, `log()`, `random()`, etc. 

#### Example Input

```
${math 2 + 2}
${math round(10/3)}
${math random(1,10)}
```

#### Example Output

```
4
3
7.525986609745358
```
